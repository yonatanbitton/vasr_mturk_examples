Instructions


 Given three images, A, A' and B, You will need to answer:
 Which candidate (B') out of {C1, C2, C3, C4} is similar to B the same way A' is similar to A?

 To answer it, you'll need to choose a candidate:

  1.   (1) understand what is the key difference between A and A'
  2.   (2) map this difference to B
  3.   (3) Verify: B → B' has the same difference as A → A'


 The main difference between the images, can be one of the semantic roles in the image: (1) verb; (2) agent; (3) item; (4) vehicle; (5) victim; (6) tool; (7) food;


 Agent examples:


 The agent answers the question "Who is doing something in the image". For example in image A' below, the agent is a cat.


 Agent - Example 1:


 Difference A,A': A child urinating, changed to cat urinating. agent changed: child → cat
 Map difference to B: child lying in a hospital bed → cat in a hospital bed
 Answer: C4 (It contains: cat in a hospital bed)
 Additional explaination: Why difference A, A' is child → cat and not female child → cat? Answer: B should contain the same agent as A. They both contain a child.

 Agent - Example 2:
